เลือกเฉพาะข้อมูลที่อยากให้ทีมหน้างานช่วยตรวจเพิ่ม รายการนี้เป็นคำสั่งงาน ไม่ใช่ค่าที่ยืนยันแล้ว
ทิศทาง social / on-camera
ส่วนนี้ใช้กำหนดน้ำเสียง จุดขาย และช็อตที่อยากเน้นสำหรับคอนเทนต์ social หลังจากสรุปชุดลงหน้างานหลักแล้ว
ประโยคเปิดหรือประเด็นดึงความสนใจที่อยากใช้เป็นตัวนำ
เลือกเฉพาะ shot ที่อยากให้เด่นกว่าภาพอื่นในคอนเทนต์
สิ่งที่คนเล่าควรพูดให้ครบเมื่อถ่ายคลิปหรือเก็บเสียงหน้างาน
โทนหรือมุมของข้อความประกอบโพสต์ ไม่ต้องเขียนยาวทั้งชิ้นก็ได้
แหล่งอ้างอิงและภาพประกอบ
เก็บเฉพาะแหล่งอ้างอิงที่ตั้งใจส่งต่อจริง และภาพตัวอย่างที่ช่วยให้ทีมหน้างานหรือทีมเขียนตามตรวจได้
ภาพอ้างอิงจาก assets ที่เลือกไว้
ใช้เฉพาะรูปที่ถูกเลือกส่งเข้า Agent เป็นตัวอย่างหรือแนวทางสำหรับทีมหน้างาน
ใส่เมื่อมีภาพอ้างอิงจากเว็บภายนอกที่อยากแนบไปด้วย
ถ้างานนี้เคยถูกมอบหมายไว้แล้ว ให้ผูกกับ assignment เดิมได้
ใช้เมื่ออยากเชื่อมชุดนี้กับงานเขียนที่มีอยู่แล้ว
