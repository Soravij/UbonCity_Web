# Frontend Project Policy

See [../PROJECT_POLICY.md](../PROJECT_POLICY.md) for the canonical project-wide policy.

## Frontend-specific rules

- Frontend consumes approved published taxonomy/content only.
- Frontend must not consume assignment requested-check drafts or Work Return internals.
- Frontend must not resolve category defaults, mapping, or AI taxonomy suggestions client-side.
- No frontend runtime behavior changed in the current CTA/Curation Work Return patch.
