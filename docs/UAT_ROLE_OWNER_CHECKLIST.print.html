1) ความพร้อมก่อนเริ่ม

  • เข้า admin-test และ collector-test ได้
  • ผ่าน Cloudflare Access OTP ได้
  • login owner ได้
  • service หลักรันครบ
  • เตรียม item / assignment / evidence พร้อม

2) Global Smoke

  • login สำเร็จ
  • logout สำเร็จ
  • session หมดอายุแล้วกลับ login ถูก
  • deep-link / return_to ถูก
  • ไม่มี auth redirect loop

3) Owner Core Flow

  • สร้างงานจาก raw/source ได้ 1 รอบ
  • แก้ metadata สำคัญก่อนสร้าง content ได้
  • สร้าง content item สำเร็จ
  • workflow ไปถึงสถานะที่ backend ควรเห็น
  • backend แสดงผลตรงกับ source/content

4) Assignment + Handoff

  • assign งานให้ editor ได้
  • assign งานให้ freelance ได้
  • เห็นงานส่งกลับจาก editor ได้
  • เห็นงานส่งกลับจาก freelance ได้
  • revision request loop ทำงานได้

5) Publish / Preview

  • review preview เปิดได้
  • publish / release action ใช้งานได้ถ้าอยู่ใน scope
  • public page ตรงกับ approved content
  • media แสดงผลถูกต้อง

6) Regression Hotspots

  • owner-only surfaces ยังเข้าได้ครบ
  • direct URL ไม่พา role boundary พัง
  • asset ไม่ข้าม item / role
  • timeline / history ตรงกับ action จริง
